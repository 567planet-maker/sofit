# 견적 요청 스튜디오 리디자인 — 개발 핸드오프

대상 파일: `quote-studio.html`

## 1. 레이아웃 — 사이드바 방식 변경
- 기존: 좌(분야)·중(작성)·우(요약) 3컬럼이 하나의 그리드 안에서 함께 스크롤되는 구조
- 변경: **앱쉘 구조**로 전환
  - 좌측 분야 패널: 폭 288px, 화면 좌측 끝에 고정, `border-right: 1px solid var(--border)`
  - 우측 요약 패널: 폭 324px, 화면 우측 끝에 고정, `border-left: 1px solid var(--border)`
  - 가운데 작성 영역: `flex:1`로 남는 공간 전부 차지, 내부 콘텐츠는 최대 860px 폭으로 센터 정렬 (`max-width:860px; margin:0 auto`)
  - 세 영역 모두 `overflow-y:auto`로 **독립 스크롤**
- 모바일(<1024px)에서는 기존과 동일하게 상단 슬라이딩 pill 탭(분야/작성/요약) + 세로 스택으로 전환

## 2. 스크롤바 디자인
- 기본 브라우저 스크롤바 대신 커스텀 스타일 적용
- 평소엔 투명(숨김), 컨테이너에 마우스 hover 시에만 얇고 둥근 thumb 표시
- thumb에 다시 hover하면 더 진한 색으로 변경(그랩 상태 피드백)
- Firefox(`scrollbar-width/color`)·Webkit(`::-webkit-scrollbar-*`) 양쪽 대응

## 3. 색상 — 빨간색(danger) 변경
- 기존: `--danger: #E5484D` / `--danger-tint: #FDECEC`
- 변경: `--danger: #E5484D` / `--danger-tint: #FDECEC`
- 적용 범위: 필수 항목 표시, 미입력 카운트 뱃지, 유효성 오류 메시지, 제출 실패 배너
- CSS 변수 하나만 교체하면 전체 반영됨 (`:root` 선언부)

## 4. 이모지·아이콘 제거
다음 요소에서 이모지/SVG 아이콘을 전부 제거하고 텍스트만 남김:
- 자동저장 안내 문구 (💾 제거)
- 제출된 견적 공개 안내 (📢 제거)
- 첨부 영역 경고 문구의 SVG 삼각형 아이콘
- 사진/도면 업로드 드롭존의 SVG 아이콘
- 개인정보 안내 문구의 자물쇠 SVG 아이콘
- 분야별 "분야 제거" 버튼의 X SVG 아이콘
- 유효성 검사 실패 배너의 원형 경고 SVG 아이콘

## 5. 가독성 개선 (선행 작업)
- 타이포 스케일 확대: 페이지 타이틀 27px, 인풋 높이 40px → 44px, 라벨/본문 12.5–14.5px
- "필수" 표시를 빨간 알약 뱃지에서 조용한 회색 텍스트로 변경
- 카드 패딩·섹션 간격 정리
