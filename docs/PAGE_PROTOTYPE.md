## 1. 공통 레이아웃 & 컴포넌트

### 1-1. `Header` (전체 공통)

```
┌─────────────────────────────────────────────────────────────────┐
│  [소핏 로고]          [포트폴리오] [서비스 소개]   [🔔 3] [메뉴▼] │
└─────────────────────────────────────────────────────────────────┘
```

| 요소 | 설명 | 노출 조건 |
|------|------|-----------|
| 로고 | 텍스트 또는 SVG, 클릭 시 `/` 이동 | 항상 |
| 포트폴리오 | `/portfolios` 링크 | 항상 |
| 서비스 소개 | 랜딩 해당 섹션으로 스크롤 앵커 | 비로그인 |
| 알림 벨 `🔔` | 읽지 않은 알림 수 뱃지 (빨간 원), 클릭 시 알림 드롭다운 | 로그인 후 |
| 유저 메뉴 `▼` | 이름 표시, 드롭다운: 내 정보 / 로그아웃 | 로그인 후 |
| 견적 요청하기 | CTA 버튼 (초록), 클릭 시 `/customer/request` 또는 `/login` | 비로그인 |

#### `NotificationBell` 드롭다운

```
┌───────────────────────────────┐
│  알림                  모두 읽음 │
├───────────────────────────────┤
│ 🔵 [새 견적서 도착] 3분 전      │
│    "강남 카페 쇼파 견적서가..."  │
├───────────────────────────────┤
│ ○  [상태 변경: 시공 중] 2시간 전│
├───────────────────────────────┤
│         전체 알림 보기          │
└───────────────────────────────┘
```

- 최대 5개 미리보기, 읽지 않은 건은 파란 점(●)
- 클릭 시 `read_at` 업데이트 + 해당 `link` 페이지 이동
- "모두 읽음" 버튼: 전체 `read_at` 일괄 업데이트
- 데이터 소스: `notifications` 테이블 (Supabase Realtime 구독)

### 1-3. `FileUploader`

```
┌─────────────────────────────────────────────┐
│                                             │
│   📎 파일을 드래그하거나 클릭하여 업로드      │
│   JPG, PNG, PDF / 최대 10MB                │
│                                             │
└─────────────────────────────────────────────┘
│ 업로드된 파일 목록:                           │
│ 🖼 매장사진_01.jpg  (2.3MB)  [미리보기] [✕]  │
│ 📄 도면.pdf         (5.1MB)           [✕]  │
│ ▓▓▓▓▓▓░░░ 업로드 중 65%                     │
```

- 드래그앤드롭 + 클릭 업로드 두 방식 모두 지원
- 이미지는 썸네일 미리보기 표시
- 업로드 진행률 프로그레스 바
- 파일 개별 삭제 버튼(✕)
- 최대 파일 수, 크기, 확장자 props로 주입
- 실패 시 빨간 에러 메시지 + 재시도 버튼

### 섹션 1 — Hero

```
┌─────────────────────────────────────────────────────────────┐
│  [헤더]                                                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   쇼파·빌트인 발주,                                          │
│   이제 소핏으로 한 번에                                       │
│                                                             │
│   부산·경남 검증 공장 · 투명한 견적 · 소핏이 끝까지 함께      │
│                                                             │
│   [무료로 견적 요청하기 →]    [공장 포트폴리오 보기]           │
│                                                             │
│   [배경: 매장 쇼파 설치 완료 사진 또는 일러스트]              │

```

| 요소 | 타입 | 설명 |
|------|------|------|
| 헤드라인 | `h1` | 굵은 폰트, 2~3줄 |
| 서브 카피 | `p` | 3가지 핵심 가치 bullet |
| CTA 버튼 (주) | `button` | 초록, 클릭 시 로그인 → `/customer/request` |
| CTA 버튼 (부) | `a` | 아웃라인, `/portfolios` 이동 |
| 배경 이미지/영상 | `img` | 쇼파 완공 사진 or 비디오 루프 |

## 부록: 공통 인터랙션 규칙

| 상황 | 처리 방식 |
|------|-----------|
| 비로그인 → 보호된 페이지 접근 | `/login?redirect=원래URL` 리디렉트 |
| 잘못된 role로 접근 (고객→공장 페이지) | 403 페이지 또는 본인 대시보드로 리디렉트 |
| 폼 유효성 오류 | 해당 필드 빨간 테두리 + 에러 메시지 인라인 표시 |
| API 요청 중 | 버튼 비활성화 + 로딩 스피너 |
| API 오류 | 상단 toast 알림 (빨강, 3초 후 사라짐) |
| 성공 완료 | 상단 toast 알림 (초록, 2초 후 사라짐) |
| 삭제 액션 | 확인 모달 ("정말 삭제하시겠습니까?") |
| 파일 업로드 실패 | 인라인 에러 + 재시도 버튼 |
| 인터넷 끊김 | 배너 경고 "연결이 끊어졌습니다. 재연결 중..." |
