분야별 폼 스키마 레퍼런스

가운데 "견적 작성" 영역의 27개 시공 분야 폼을 스키마 구동(schema-driven) 구조로 렌더링한 정적 레퍼런스입니다. 분야마다 컴포넌트를 따로 만들지 않고, FIELD_SCHEMAS 데이터 + 공통 렌더러(renderField) 하나로 27개 전부를 그립니다. 폼의 생김새를 바꾸려면 렌더러를, 분야별 내용을 바꾸려면 스키마 데이터를 수정하세요.
